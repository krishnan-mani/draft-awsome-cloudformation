Pre-requisites:

- You can progress through this workshop by checking out a branch that corresponds to each story (or stories)

For example:

```
$ git branch
  Story-1
  Story-2
  Story-3
  Story-4
  Story-5
  Story-6
  Story-7
  Story-8
* develop
$ git checkout Story-1

```

- Look at stories.md for the description of stories
